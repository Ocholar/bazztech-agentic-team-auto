# Bazztech AI SaaS: Project Charter

## Objective

To build a "low-effort, self-running" SaaS platform that provides autonomous AI Digital Employees to Kenyan MSMEs, managed via a Next.js portal and monetized via Equity Jenga API.

## Core Phases (Completed)

1. **Pivot Dashboard**: Converted the legacy tracking app into an authenticated multi-tenant SaaS portal.
2. **Jenga API Polling**: Replaced manual M-Pesa tracking with automated bank statement polling.
3. **n8n Cloud Migration**: Moved the AI orchestrator to PikaPods for 24/7 reliability.
4. **Onboarding & Billing**: Implemented tiered MSME pricing and a 3-step setup wizard.

## Tech Stack

- **Frontend**: Next.js 14 (App Router)
- **Database**: Supabase (PostgreSQL + Vector)
- **Automation**: n8n
- **Payments**: Equity Bank Jenga API
